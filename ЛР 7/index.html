Основные приёмы и методики

Добро пожаловать! Поговорим о практике создания адаптивных интерфейсов, необходимых инструментах и шаблонах.

Зачем нужна адаптивность?

  • Рост мобильного трафика
  • Удобство для всех устройств
  • Плюс в SEO за "mobile-friendly"
Обсудим статистику мобильного использования и влияние на поисковую выдачу.

Meta viewport
Этот метатег задаёт масштаб страницы и ширину области просмотра под устройство.

Flexbox & Grid

Быстрое выравнивание и сложные макеты


.container-flex {
  display: flex;
  gap: 10px;
}
.container-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
}
Flexbox удобен для одномерных раскладок, Grid — для двумерных шаблонов с рядами и колонками.

Медиа-запросы


@media (min-width: 768px) {
  .content {
    padding: 20px;
  }
}
Используем breakpoints от мобильных к десктопным, ориентируясь на контент.

Подход Mobile-First

  1. Базовые стили для смартфонов
  2. Добавляем стили для планшетов
  3. И завершаем стили для больших экранов
Сначала лёгкая версия для мобильных, потом улучшаем для планшетов и десктопов.

Дополнительные методы

  • Относительные единицы: rem, vw, %
  • CSS-фреймворки: Foundation, Bulma
  • Эмуляция устройств в DevTools
